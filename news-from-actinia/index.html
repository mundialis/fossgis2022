#### actinia-module-plugin Selbstbeschreibung * Listet alle GRASS GIS und actinia Module per HTTP GET `/modules` * Filter sind verfügbar: * family - Typ des GRASS GIS Moduls ["d", "db", "g", "i", "m", "ps", "r", "r3", "t", "test", "v"] * tag - Filter für Kategorien * category - Noch ein filter für Kategorien * record=full - Zeigt die vollständige Beschreibung für die Liste an (gecached) * Bestimmtes Modul über ID anfragen, z.B. HTTP GET `/modules/r.slope.aspect` --- { "id": "r.slope.aspect", "categories": [ "aspect", "curvature", "grass-module", "raster", "slope", "terrain" ], "parameters": [ { "description": "Name of input elevation raster map.
