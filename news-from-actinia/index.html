#### actinia-module-plugin self-description * get all GRASS GIS and actinia modules by HTTP GET `/modules` * filters are available: * family - Type of GRASS GIS module ["d", "db", "g", "i", "m", "ps", "r", "r3", "t", "test", "v"] * tag - Filter for categories * category - Another filter for categories * record=full - Shows full record description for list.
